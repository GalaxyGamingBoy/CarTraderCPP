## Build

### Windows

- Download the repo
- Unzip the project
- Open CMD and navigate to the folder
- Type `build.bat`
- Done

### Linux

- Download the repo
- Unzip the project
- On a command prompt and navigate to folder
- Type `bash build.sh`
- Done
